## 01. Number Type  
Reads an integer from the console and determines its sign.  
- Prints `"negative"` if the number is less than zero.  
- Prints `"positive"` if the number is greater than zero.  
- Prints `"zero"` if the number is exactly zero.

## 02. ATM  
Simulates a basic ATM withdrawal process by checking the account balance and the daily transaction limit.  
- Reads three integers: **balance**, **withdraw** (requested amount), and **limit**.  
- Evaluates the inputs and prints:  
  - `"The withdraw was successful."` if both the balance and the limit are sufficient to cover the requested amount.  
  - `"The limit was exceeded."` if the withdrawal amount is greater than the daily limit.  
  - `"Insufficient availability."` if the account balance is less than the requested amount.

## 03. Biggest of Five Numbers  
Reads five integer numbers from the console and identifies the largest one.  
The program compares all five inputs and prints the maximum value.

## 04. Number Operations  
Reads two real numbers and a mathematical operator (`+`, `-`, `*`, `/`) from the console.  
The program performs the requested operation and prints the result in the format: `{N1} {operator} {N2} = {result}`, with the result formatted to the second decimal place.
